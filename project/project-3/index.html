Iron Quest

Quantified Self

Project Details

  • Role & Duration : Data Visualization creator | May 2020
  • What I Did : Used the Pomodoro technique as a data collection method to quantify my learning progress and visualize the results
  • Tools Used : Be Focused Pro app, Google Sheets, Tableau
  • Skills : Data Collection / Data Preparation / Data Transformation / Data Visualization
  • Lessons Learned : Data Collection | Creating Data Set | Sharing the Work
  • Impact/Outcome : Tableau visualization that quantified learning with the Pomodoro Technique. Publishing these learnings in Free Code Camp

Overview

“Iron Quest is a monthly community-led data visualization project which follows a similar format to the Tableau Iron Viz feeder competitions. Participants have a calendar month to source a suitable data set and then design, build and submit a data visualization. Every submission should use a data set that somehow touches upon the chosen theme for that month and every viz should aim to showcase best practices & skill in design, storytelling & analysis.” - Sarah Bartlett (Tableau Zen Master) Iron Quest challenges are done on a monthly basis and the themes are chosen by Sarah Bartlett.

Challenge

The May Iron Quest theme was Quantified Self. “For much of this time we were battling the global Coronavirus pandemic, with Government imposed lockdowns in place across the world. As a result, daily routines were turned upside-down and many people were juggling working from home at the same time as caring for children or relatives. With these things in mind, I thought it would be an interesting time to start tracking data about our lives and visualizing the results.” - Sarah Bartlett

Data Collection

The challenges are great because each one that I’ve participated in has required me to create my own data set. This is super helpful to develop skills like data sourcing and data preparation.

This Iron Quest challenge required each participant to create a data set based on their activities during the pandemic. After brainstorming I decided to think about how to quantify learning. I had just found learning platforms like Data Camp and had recently subscribed to take learning tracks. I thought it would be interesting to quantify this learning process and create a data visualization.

Solution

Be Focused Pro

Increased time at home offers an opportunity for growth and learning. When the pandemic began, I started to review how I was spending my extra time at home. I tracked this data, established a learning routine, and finally visualized the results. I did this by using the Pomodoro technique. Several apps exist that help you use the Pomodoro Technque. I used the “Be Focused Pro” app. The greatest benefits are that you can enter a name for your task, create tags and notes, and add a “due date.”

This data helps you understand the amount of time spent completing a project or goal. You can download a csv of your data, which includes a time stamp, the name of the activity, the duration of your pomodoro task, and metadata.

The app gives you the ability to download a csv – this is a great feature because you can transform your own data. This allows you to create a data set that best suits your purposes. You can understand your learning patterns, track progress, and visualize the various components.

Data Exploration

Before I begin to clean up and transform this data, I asked a few questions. These questions guided the data transformation & data preparation.

  1. What am I most interested in understanding about this data?
  2. Do I want to understand how long it takes me to complete a learning goal?
  3. Am I interested in the length of time required to complete the various phases of my goals?
  4. Am I more interested in seeing the progress of my various goals?
  5. Do I want to build something that serves more as motivation? Should I focus on what I’ve accomplished and the goals I am working towards?
  6. What time frame do I want to focus on and how much granularity do I need? Minutes, months, or days?
  7. Do I want to look at the time of day that I do the most Pomodoro activities? Will this knowledge help me organize my schedule so that I can focus during my productive hours?

Data Transformation & Visualization

After completing the exercise I spent time cleaning the data in the csv. I focused on the time stamp and created new columns of data, which included: Month, Day, Year, Minutes

I also spent time creating new columns of data which included: Type, Learning Platform, Learning Skill, Course Title,

Finally, I thought about the visualization I wanted to create in Tableau – the waffle chart. This visualization required additional columns: Row, Column, Progress, Minutes

There were several visualizations created as I attempted to answer my questions from the data exploration. I began to think about the dashboard design and was again guided by the questions. The final visualizations chosen were those that provided the most clarity to my questions.

Lessons Learned

Data Collection & Creating Data Set

Iron Quest is probably my favorite data visualization challenge because it’s so fun to work with custom data sets in Tableau, learning how to create data sets from scratch, and applying new tools/methodologies. This challenge helped me understand that I needed to take a step back and reflect on my purpose. The reflection helped me understand what I needed to do to structure my data best for Tableau.

Share Your Work

This visualization is all about learning. It focuses on what I’m learning, and is a tool that I use to analyze my own learning patterns. But this is something that anyone can adapt and make it better to meet their needs. I wanted to share this project with others and applied to be a writer in freeCodeCamp. I pitched the idea for an article about this project and received help from the editors at freeCodeCamp.
